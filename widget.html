    Install Latest Firmware on your NodeMCU or ESP8266

    1. NodeMCU comes with the Lua firmware pre-loaded. ESP8266's don't have Lua preloaded. To load the latest firmware version you should build your own at NodeMCU-build.com or you can download a tested working version from this workspace:
      • Full nodemcu-master-13-modules-2016-01-17-01-29-27-float.bin
      • Basic nodemcu-master-10-modules-2016-01-10-08-50-08-integer.bin
    2. You can flash the firmware by using nodemcu-flasher on Windows or Python Esptool on Linux. You can also try the flasher tools from Espressif Flash Download Tools page.
